# What is it
![Grafana](https://i.imgur.com/JNPAJ5B.png)
This grafana dashboard trends the data relevant to my KavaNest system, including
* Temparatures
* Radiator valves
* Heating status (on or off)
* Cluster & other raspberry pi monitoring
